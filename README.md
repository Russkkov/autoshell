# AutoSHELL

Script en Bash para crear automáticamente reverse Shell especificando únicamente la IP y el puerto a utilizar.

 

Tipos de reverse Shell incluidos:

- Awk
- Bash
- C
- C#
- Dart
- Golang
- Groovy
- Haskell
- Java
- Javascript
- Lua
- Nc
- Ncat
- Node.js
- Perl
- PHP
- PowerShell
- Python
- Ruby
- Rustcat
- Socat 
- Telnet
- Windows ConPty
- Zsh
